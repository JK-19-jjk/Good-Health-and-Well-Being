Student 2 - Website Designer & Developer

Home Page

Technical Description

The Home Page serves as the introduction to the website. It includes a navigation bar, an interactive image slideshow, a mission statement overlay, and four interactive cards linking to key pages. CSS was used for styling, hover effects, and layout structure, while HTML ensured proper content structuring.

Accessibility

Alt attributes were added to images for screen readers, and sufficient contrast was maintained in colors for readability. Semantic HTML was used to enhance accessibility.

Table Page

Technical Description

The Table Page contains a structured, interactive table related to the website’s theme. It includes at least 8 rows and 4 columns, alternating row colors for readability, and tooltips for additional information. CSS provides hover effects and table styling.

Accessibility

ARIA labels were used for screen reader support, and table headers (th elements) help organize data. Contrast and font size ensure legibility.

Content Page

Technical Description

The Content Page includes structured headings, paragraphs, internal navigation links, and interactive elements. A "Go to Top" button ensures easy navigation, and CSS styles maintain a consistent theme.

Accessibility

Headings follow a logical hierarchy, alt text is provided for images, and keyboard navigation is supported.

Challenges and Lessons Learned

One challenge was ensuring the CSS styles remained consistent across all team members' pages. Collaboration and effective use of a global CSS file helped maintain uniformity. Another challenge was implementing interactive hover effects for the homepage elements, which required careful positioning and responsiveness testing.

Compliance

The pages follow web accessibility guidelines by using semantic HTML, appropriate color contrast, and ARIA labels. Code validation ensures compliance with web standards.
